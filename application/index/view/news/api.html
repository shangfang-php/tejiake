  • 1.商品查询接口
接口地址：http://www.tejiake.com/api/goods
支持格式：json
请求方式：post
请求示例：sign=c0964a2f5d8198e11ac8872d061df0fc&user_id=1&timestamp=13333333333&type=1&page=1
请求参数说明：
名称 类型 必填 说明
  user_id int 是 特价客注册时分配的用户ID
  timestamp String 是 时间戳
  sign string 是 加密签名，加密格式为md5(appkey+md5(timestamp)) + 只表示连接符,不参与加密
  type int 是 商品类型 1爆款单 2限时抢购 3过夜单 4直播单 5视频单
  page int 否 分页获取数据（每页100条）
